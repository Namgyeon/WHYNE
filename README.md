## 📍 팀원별 역할 분담

- 공용 컴포넌트 제작 -> 페이지 제작 -> 오류 테스트 -> 발표 자료 준비
<br><br>


### 🧑‍💻 남기연

- 발표 자료 준비
- 페이지 : 와인 상세 페이지
- 공용 컴포넌트 : Modal - 리뷰 등록/ 수정, Icon, Card - mylist, Card - detail
- 기능
  - 아이콘 컴포넌트 화
  - 와인 상세 정보와 리뷰 목록 출력
  - 리뷰 등록/수정 클릭시 리뷰 등록/수정 기능을 가진 모달 생성
  - 리뷰 수정시, 날짜 입력 옆에 "(수정됨)" 텍스트 추가
  - 기본 alert 창 대신 toast 기능 구현
